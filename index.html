How it works

Line mode: click one grid intersection to start a segment, then click a second point to place it.

Lasso mode: drag a freehand loop around plotted lines to select them, then drag the selection box to move them.

Use Ctrl/Cmd + C and V to copy and paste the selected lines.

Text mode: click anywhere on the sheet and start typing directly on the page.

Select a label to edit it again, or drag the handle above it to rotate.

Everything snaps to the isometric lattice so notes and lines stay aligned to the sheet.

Use Esc to cancel a pending segment.
